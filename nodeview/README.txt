
NodeView README
===============

by Andrew Apted.  21st May 2007.


@@@ Intro


COLORS
------

purple   : partition lines / highlighted subsector
sky blue : minisegs

bright white  : one-sided line
bright yellow : 2S line marked impassible
bright red    : 2S line on a closed door

bright orange : gap too narrow for player to pass
bright green  : dropoff too big for player to climb

gray : any other two-sided line


KEYS
----

mouse click : go down one side of the BSP tree,
              highlight the subsector when reached

CTRL+click : instantly highlight the subsector
             (same as t following by clicking all the way down)

u : go Up the BSP tree
t : go back to Top of BSP tree

p : toggle Partition drawing (below/on-top/none)
m : toggle seg Mode (all/normal-only)
g : toggle Grid

+/- : zoom in / out
